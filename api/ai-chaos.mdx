## cruel.ai.rateLimit()

Simulates AI API rate limiting.

```typescript
function rateLimit<T extends AnyFn>(
  fn: T,
  opts: number | { rate: number; retryAfter?: number } = 0.1
): T
```

<ParamField path="fn" type="T extends AnyFn" required>
  The function to wrap
</ParamField>

<ParamField path="opts" type="number | object" default="0.1">
  Rate limit configuration. Can be a number (rate) or object with options
  
  <Expandable title="object properties">
    <ParamField path="rate" type="number" required>
      Rate at which to trigger rate limit (0-1)
    </ParamField>
    <ParamField path="retryAfter" type="number" default="60">
      Retry-After value in seconds. Default is 60
    </ParamField>
  </Expandable>
</ParamField>

**Returns:** `T` - Function that throws `CruelAIError("rate_limit")` at the specified rate

### Example

```typescript
import { cruel } from '@cruel/cruel'

const limitedAI = cruel.ai.rateLimit(async (prompt: string) => {
  return openai.chat(prompt)
}, { rate: 0.1, retryAfter: 120 }) // 10% rate limited, retry after 120s
```
